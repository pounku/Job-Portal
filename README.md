# Job-Portal
Job Portal
Description

A Job Portal is a web application that connects job seekers with employers. It allows users to search and apply for jobs, while employers can post job openings and manage applications. The project is designed with a user-friendly interface and basic job portal functionalities.

Features
User Registration and Login
Browse Available Jobs
Search Jobs by Title or Category
Apply for Jobs
Employer Job Posting
View Job Listings
Responsive User Interface
Technologies Used
HTML
CSS
JavaScript
VS Code
